## 变更记录说明

首个版本的功能将标记为 `v$version` `添加`；

后期新增加的功能将标记为 `v$version` `新增`；

后期修改的功能将被追加为 `v$version` `修改`；

后期被作废的功能将标记为 `v$version` `作废` 并会标注删除线；

后期被删除的功能将标记为 `v$version` `移除` 并会标注删除线。
